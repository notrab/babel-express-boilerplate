Get started writing ES2015 in your Express apps and leave the compiling to [Babel](https://babeljs.io/).

## Installation
    git clone https://github.com/notrab/express-babel-starter
    cd express-babel-starter
    npm install
    cp .env.sample .env

### Run in Development
    npm start

### Deploy to Heroku
    git push heroku
